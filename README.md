# Quick start

`pip install -r .\requirements.txt`

### Base user:

- username: `user1`
- password: `1`

## Features:

### Extras:

- Categories for notes
- Icon based on category
- Notes colors
- Sorting by date and name, click arrow down
- User settings, can change password, username
- Sign up
- Passwords are hashed

### Minimum:

- Auth
- Note creation
- Note overview
- Note Editing
- Note Deleting
